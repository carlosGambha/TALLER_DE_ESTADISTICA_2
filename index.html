ipak <- function(pkg){
  
  # Definir repositorio CRAN (evita el error del mirror)
  options(repos = c(CRAN = "https://cloud.r-project.org"))
  
  # Identificar paquetes instalados
  installed <- rownames(installed.packages())
  
  # Identificar paquetes faltantes
  new.pkg <- setdiff(pkg, installed)
  
  # Instalar paquetes faltantes
  if(length(new.pkg) > 0){
    install.packages(new.pkg, dependencies = TRUE)
  }
  
  # Cargar paquetes sin mostrar mensajes
  invisible(lapply(pkg, function(p){
    suppressPackageStartupMessages(
      library(p, character.only = TRUE)
    )
  }))
}

2 Introduccion

Tablas de frecuencias unidimensionales y tablas de contingencia

Enfoque estad√≠stico y anal√≠tico

Las tablas de frecuencias son herramientas fundamentales de la estad√≠stica descriptiva porque permiten organizar, resumir e interpretar datos. Desde una perspectiva anal√≠tica, cumplen una funci√≥n clave: transformar datos crudos en informaci√≥n estructurada, facilitando la identificaci√≥n de patrones, tendencias y relaciones entre variables.

Tablas de frecuencias unidimensionales

Una tabla de frecuencias unidimensional resume la distribuci√≥n de una sola variable XXX, ya sea cualitativa o cuantitativa.

Su objetivo es responder:

  • ¬øQu√© valores toma la variable?

  • ¬øCon qu√© frecuencia aparecen?

  • ¬øC√≥mo se distribuyen proporcionalmente?

  • ¬øC√≥mo se acumulan los datos?

Definici√≥n

Sea una variable estad√≠stica XXX con kkk categor√≠as o valores distintos:

x1,x2,x3,‚Ä¶,xkx_1, x_2, x_3, \dots, x_kx1‚Äã,x2‚Äã,x3‚Äã,‚Ä¶,xk‚Äã

y un total de observaciones:

NNN

La tabla de frecuencias organiza cada valor xix_ixi‚Äã con sus respectivas frecuencias.

Tipos de frecuencias (unidimensional)

a) Frecuencia absoluta (fi o ni)(f_i \text{ o } n_i)(fi‚Äã o ni‚Äã)

Es el n√∫mero de veces que aparece un valor xix_ixi‚Äã en el conjunto de datos.

fi=nif_i = n_ifi‚Äã=ni‚Äã

Interpretaci√≥n:

Propiedad:

‚àëi=1kfi=N\sum_{i=1}^{k} f_i = Ni=1‚àëk‚Äãfi‚Äã=N

Es decir, la suma de todas las frecuencias absolutas debe ser igual al total de datos.

  1. Frecuencia relativa (hi)(h_i)(hi‚Äã)

Es la proporci√≥n de veces que aparece un valor respecto al total de observaciones.

hi=fiNh_i = \frac{f_i}{N}hi‚Äã=Nfi‚Äã‚Äã
